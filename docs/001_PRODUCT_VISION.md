# Product Vision

## Product statement

For software developers and engineering organizations that need to deliver systems faster without losing control, Nova Platform is an AI-native software engineering platform that combines a browser IDE, project administration, specialized agent teams, project memory, secure execution, and governed automation. Unlike editor-only assistants, Nova coordinates the full engineering lifecycle and records every important decision and action.

## User outcomes

A user should be able to:

1. Create or import a repository.
2. Describe a feature, bug, or system requirement in Arabic or English.
3. Receive an implementation plan with risks and acceptance criteria.
4. Approve a controlled agent workflow.
5. Watch agents inspect code, modify files, execute tests, and produce a pull request.
6. Review a complete timeline of reasoning summaries, tool calls, diffs, tests, costs, and approvals.
7. Submit product feedback that is classified and routed to the correct internal agent team.

## Core product surfaces

### Web administration

- Organizations, teams, roles, and permissions
- Projects, repositories, environments, and integrations
- AI providers, models, budgets, and usage policies
- Agent registry and workflow configuration
- Feedback, analytics, audit logs, and release management

### Web IDE

- File explorer
- Monaco Editor
- Project search
- Integrated terminal
- Git changes and diffs
- AI chat and task panel
- Preview and test output
- Workflow timeline

### Agent operating layer

- Agent registry
- Orchestrator and planner
- Tool gateway
- Workflow engine
- Event bus
- Project memory and knowledge retrieval
- Policy and permission engine
- Evaluation, retry, and escalation

## Differentiators

- Feedback-to-fix automation built into the product
- Multiple specialized agents rather than one generic assistant
- Human approval gates configurable by organization and risk level
- Strong enterprise support for private deployment and local models
- Deep support for enterprise stacks, initially Angular, Spring Boot, Java, SQL, Oracle, PostgreSQL, Docker, and GitHub
- Arabic and English product experience
- A transparent timeline for every AI action

## Product promise

Nova will not claim that generated code is correct merely because a model produced it. A task is considered complete only when the configured evidence exists, such as successful compilation, automated tests, security checks, and required human approvals.
