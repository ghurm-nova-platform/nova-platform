CI Observation Agent

This agent observes CI only. It never reruns, approves, merges, or deploys.

Fetches GitHub Actions workflow runs for a successful Pull Request operation and surfaces overall health, failed jobs/steps, and retry guidance. Read-only — no workflow triggers or deployment actions.
