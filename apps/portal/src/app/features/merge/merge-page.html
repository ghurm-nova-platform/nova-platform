Merge Agent

Merge Agent performs repository merge only after successful Approval Gate validation.

Validates an approved Approval Gate decision and matching evidence fingerprints; merges the associated Pull Request via the configured provider; verifies post-merge state. Does not deploy, rerun CI, or bypass Approval Gate.
